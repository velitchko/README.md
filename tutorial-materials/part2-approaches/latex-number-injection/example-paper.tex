\newcommand*{\percentageRounded}[2]{%
	\pgfmathparse{#1*100/#2}%
	\pgfmathprintnumber[fixed, precision=1]{\pgfmathresult}%
}

\noindent My first key metric is \KeyMetricOne{} and the second is \KeyMetricTwo. The sum of the two is \pgfmathparse{\KeyMetricOne+\KeyMetricTwo}\pgfmathprintnumber[fixed, precision=0]{\pgfmathresult}, and the first is $\approx$\percentageRounded{\KeyMetricOne}{\KeyMetricTwo}\% of second. The key metric for 2026 is \KeyMetricInMMXXVI, or rounded to 3 digits it is \pgfmathparse{\KeyMetricInMMXXVI}\pgfmathprintnumber[fixed, precision=3]{\pgfmathresult}. A metric that contains a string is also possible, such as ``\KeyMetricString''.
